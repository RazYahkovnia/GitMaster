# GitMaster

GitMaster is a VS Code extension that provides advanced git features to enhance your development workflow.

## Features

- **File History Viewer**: View all commits that modified the currently open file
  - See commit messages, authors, dates, and hashes
  - Click on any commit to view the diff
  - Automatically updates when switching between files

## Usage

1. Open a file in a git repository
2. Click on the GitMaster icon in the activity bar
3. View the commit history for the current file
4. Click on any commit to see what changed

## Requirements

- Git must be installed and accessible from the command line
- The file must be part of a git repository

## Development

To set up the development environment:

```bash
npm install
npm run compile
```

Press F5 in VS Code to launch the extension in debug mode.
